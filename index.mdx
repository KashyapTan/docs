Xpdite is an always-on-top Windows desktop app that combines an Electron shell, a React UI, a Python backend, and MCP-powered tools. You can capture your screen, chat across isolated tabs, switch between local and cloud models, run terminal or retrieval workflows, and send conversations to mobile channels.

## Architecture at a glance

- **Electron host** manages the window, IPC, Python lifecycle, and Channel Bridge lifecycle.
- **React renderer** owns chat, tabs, settings, history, meeting recorder, and streaming UI.
- **Python backend** runs REST, WebSocket, LLM routing, persistence, scheduler, and tool execution.
- **Channel Bridge** connects Telegram, Discord, and WhatsApp to Xpdite over loopback APIs.

<Note>
  Xpdite is documented as a local-first Windows app. Internal loopback APIs and placeholder connectors are called out explicitly instead of being presented as public internet services.
</Note>
