## Core experience

<CardGroup cols={2}>
  <Card title="Chat and tabs" icon="messages" href="/features/chat-and-tabs">
    Work across isolated tabs with independent queues, screenshots, and conversation state.
  </Card>
  <Card title="Models and providers" icon="cpu" href="/features/models-and-providers">
    Use Ollama locally or connect Anthropic, OpenAI, Gemini, and OpenRouter.
  </Card>
  <Card title="Skills" icon="wand-magic-sparkles" href="/features/skills">
    Load reusable instruction bundles with slash commands or tool-driven retrieval.
  </Card>
  <Card title="Terminal" icon="terminal" href="/features/terminal">
    Let the model run commands with approval gates, PTY sessions, and streamed output.
  </Card>
</CardGroup>

## Persistent knowledge and outputs

<CardGroup cols={2}>
  <Card title="Memory" icon="brain" href="/features/memory">
    Store durable markdown knowledge under controlled filesystem roots.
  </Card>
  <Card title="Artifacts" icon="file-code" href="/features/artifacts">
    Persist generated code, markdown, and HTML outputs as standalone records.
  </Card>
  <Card title="Scheduled jobs" icon="calendar-clock" href="/features/scheduled-jobs">
    Run recurring or one-shot AI tasks through the same conversation pipeline.
  </Card>
  <Card title="Notifications" icon="bell" href="/features/notifications">

  </Card>
