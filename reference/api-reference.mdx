## Base behavior

- REST routes live under `/api/*`
- Real-time chat and streaming use `/ws`
- Internal mobile integration uses `/internal/mobile/*`
- The backend binds to loopback by default and selects the first free port from `8000` through `8009`

<Warning>
  `/internal/mobile/*` is a local integration surface for the Channel Bridge. It is not a public internet API and should remain loopback-only.
</Warning>

## Common conventions

- JSON is used unless a route says otherwise.
- FastAPI errors typically return `{ "detail": "..." }`.
- Artifact endpoints may require the `X-Xpdite-Server-Token` header.
- Marketplace endpoints use the same local-access guard pattern as artifact endpoints.

## WebSocket protocol

Message envelopes use:

```json
{
  "type": "message_type",
  "tab_id": "default",
  "content": "..."
}
```

### Client to server

- Tab lifecycle: `tab_created`, `tab_closed`, `tab_activated`
- Chat flow: `submit_query`, `retry_message`, `edit_message`, `set_active_response`
- Conversation management: `clear_context`, `resume_conversation`, `load_conversation`, `delete_conversation`, `get_conversations`, `search_conversations`
- Queue and cancellation: `stop_streaming`, `cancel_queued_item`
- Screenshots and capture: `remove_screenshot`, `set_capture_mode`
- Terminal control: `terminal_approval_response`, `terminal_session_response`, `terminal_stop_session`, `terminal_kill_command`, `terminal_set_ask_level`, `terminal_resize`
- YouTube approval: `youtube_transcription_approval_response`
- Meeting recorder: `meeting_start_recording`, `meeting_stop_recording`, `meeting_audio_chunk`, `get_meeting_recordings`, `load_meeting_recording`, `delete_meeting_recording`, `search_meeting_recordings`, `meeting_get_status`, `meeting_get_compute_info`, `meeting_get_settings`, `meeting_update_settings`, `meeting_generate_analysis`, `meeting_execute_action`
- Ollama pull: `ollama_pull_model`, `ollama_cancel_pull`

### Server to client

- Base readiness: `ready`, `error`
- Screenshots: `screenshot_start`, `screenshot_added`, `screenshot_removed`, `screenshots_cleared`, `screenshot_ready`
- Chat streaming: `query`, `thinking_chunk`, `thinking_complete`, `response_chunk`, `response_complete`, `token_usage`
- Tools and artifacts: `tool_call`, `tool_calls_summary`, `artifact_start`, `artifact_chunk`, `artifact_complete`, `artifact_deleted`
- Conversations: `context_cleared`, `conversation_saved`, `conversations_list`, `conversation_loaded`, `conversation_deleted`, `conversation_resumed`
- Queue: `queue_full`, `query_queued`, `queue_updated`, `ollama_queue_status`
- Terminal: `terminal_approval_request`, `terminal_session_request`, `terminal_session_started`, `terminal_session_ended`, `terminal_running_notice`, `terminal_output`, `terminal_command_complete`
- Meeting and notifications: `meeting_*`, `notification_added`, `notification_dismissed`, `notifications_cleared`
- Mobile stop acknowledgement: `generation_stopped`
